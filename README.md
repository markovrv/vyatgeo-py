# vyatgeo-py
Installing:

1. create local venv, pip install django, psycopg2
2. install Postgres, set vyatgeo/settings.py
3. python manage.py migrate
4. python manage.py createsuperuser
5. cd lang_frontend
6. install node.js, npm
7. npm install
8. create dir ../static
9. npm run serve
10. python manage.py runserver
